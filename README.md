# Plantio Mobile Application

## Overview

Plantio is a React Native mobile application developed for plant management and marketplace operations. The application allows users to manage plants, browse plant listings, add favorites, view weather information, and perform plant buying and selling operations through a user-friendly mobile interface.

## Features

* User Registration and Login
* JWT Authentication
* Plant Marketplace
* Plant Management
* Favorite Plants
* User Profiles
* Weather Information
* Plant Details
* Shopping Cart and Checkout
* Image Upload Support

## Technologies

* React Native
* Expo
* JavaScript
* AsyncStorage
* REST API Integration

## Backend Integration

The application communicates with a Node.js, Express.js, and MongoDB backend through RESTful API endpoints.
